# prompt-enhancer

Use this skill to transform a rough request into an operational prompt.
        Preserve intent, name the desired outcome, list assumptions, define write
        boundaries, cite source-of-truth checks, and set an objective validation
        threshold. Execute the enhanced prompt when safe; return prompt text only
        when explicitly requested.
