# Hermes Capstone Orchestrator

Orchestrates mastery projects that combine setup, safety, profiles, skills, plugins, automation, and troubleshooting.

## Tutorial level

Builder/master

## What it teaches

Capstones prove that a learner can combine Hermes primitives safely, explain tradeoffs, and validate results.

## Slash commands

- `/capstone`

## Tools

- `tutorial_plan_capstone`
- `tutorial_review_capstone`
- `tutorial_score_mastery`

## Bundled skills

- `hermes-capstone-orchestrator:mastery-capstones`

## Safety

This plugin is designed for tutorial use. It does not require secrets. Runtime learner progress, when present, stays in the active profile's local data path and is not part of public packages.

Official Hermes docs and live CLI output are the source of truth if tutorial content differs.
