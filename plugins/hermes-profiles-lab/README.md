# Hermes Profiles Lab

Teaches Hermes profiles, profile homes, working directories, aliases, and distributions.

## Tutorial level

Intermediate operator

## What it teaches

A profile has its own Hermes home; its working directory and system sandbox are separate choices.

## Slash commands

- `/profiles-lab`
- `/profile-simulator`

## Tools

- `tutorial_explain_profile_layout`
- `tutorial_compare_profile_creation_modes`
- `tutorial_plan_profile`

## Bundled skills

- `hermes-profiles-lab:profile-basics`
- `hermes-profiles-lab:profile-builder-basics`
- `hermes-profiles-lab:profile-distribution-basics`

## Safety

This plugin is designed for tutorial use. It does not require secrets. Runtime learner progress, when present, stays in the active profile's local data path and is not part of public packages.

Official Hermes docs and live CLI output are the source of truth if tutorial content differs.
