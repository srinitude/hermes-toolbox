# Hermes Tools Lab

Teaches built-in tools and toolsets through controlled exercises.

## Tutorial level

Confident beginner

## What it teaches

Tools are capabilities, not toys: use the smallest safe toolset that proves the task.

## Slash commands

- `/tools-lab`

## Tools

- `tutorial_list_tool_lessons`
- `tutorial_start_tool_lesson`
- `tutorial_grade_tool_use`

## Bundled skills

- `hermes-tools-lab:tools-overview`
- `hermes-tools-lab:file-tool-practice`
- `hermes-tools-lab:web-tool-practice`
- `hermes-tools-lab:terminal-safety-practice`

## Safety

This plugin is designed for tutorial use. It does not require secrets. Runtime learner progress, when present, stays in the active profile's local data path and is not part of public packages.

Official Hermes docs and live CLI output are the source of truth if tutorial content differs.
