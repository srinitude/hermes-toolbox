# Hermes Memory Lab

Teaches memory, user profile, session search, and external memory-provider concepts.

## Tutorial level

Intermediate operator

## What it teaches

Save durable facts in memory, retrieve past task context with session search, and keep procedures in skills.

## Slash commands

- `/memory-lab`
- `/memory-quiz`

## Tools

- `tutorial_classify_memory_candidate`
- `tutorial_memory_vs_skill_vs_session`

## Bundled skills

- `hermes-memory-lab:memory-basics`
- `hermes-memory-lab:session-search-basics`
- `hermes-memory-lab:honcho-overview`

## Safety

This plugin is designed for tutorial use. It does not require secrets. Runtime learner progress, when present, stays in the active profile's local data path and is not part of public packages.

Official Hermes docs and live CLI output are the source of truth if tutorial content differs.
