# Hermes Concept Glossary

Translates Hermes terminology into plain language for non-technical learners.

## Tutorial level

Total novice

## What it teaches

Learn core Hermes words before following procedures that depend on them.

## Slash commands

- `/what-is`

## Tools

- `explain_hermes_term`

## Bundled skills

- `hermes-concept-glossary:glossary-basics`
- `hermes-concept-glossary:analogy-library`
- `hermes-concept-glossary:do-not-confuse-cards`

## Safety

This plugin is designed for tutorial use. It does not require secrets. Runtime learner progress, when present, stays in the active profile's local data path and is not part of public packages.

Official Hermes docs and live CLI output are the source of truth if tutorial content differs.
