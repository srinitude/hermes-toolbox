# Hermes Config Lab

Teaches Hermes configuration without overwhelming users.

## Tutorial level

Confident beginner

## What it teaches

Use `hermes config` for non-secret settings and setup/auth flows for secrets.

## Slash commands

- `/config-lab`
- `/config-explain`

## Tools

- `tutorial_explain_config_key`
- `tutorial_recommend_config_path`
- `tutorial_check_config_safety`

## Bundled skills

- `hermes-config-lab:config-basics`
- `hermes-config-lab:secrets-vs-config`
- `hermes-config-lab:config-troubleshooting`

## Safety

This plugin is designed for tutorial use. It does not require secrets. Runtime learner progress, when present, stays in the active profile's local data path and is not part of public packages.

Official Hermes docs and live CLI output are the source of truth if tutorial content differs.
